В репозитории есть файл settings.py, все данные вводятся туда.

LINK - это страница, которая будет парсится
HOME_LINK - это страница, на которой будет осуществлятся авторизация на сайт
SEND_LINK - страница отправки сообщений

TIME - задержка в секундах
TEXT - текст, который будет вставлятся в сообщение

LOGIN - логин
PASSWORD - пароль


ВАЖНО!!! 
Все вводимые параметры кроме TIME обязательно должны находится между кавычками пример - PARAMETR = 'value'



Порядок применений изменений:

    heroku login
    # ввод изменений #
    git add settings.py
    git commit -m 'change_name'
    git push heroku master

Посмотреть что происходит:

    heroku logs -t
